1. Project Overview

NeuroSynth currently runs as a focused neurological risk prediction platform with a minimal local-first workflow, while preserving the larger enterprise structure for future expansion.

Current objective: Enable immediate end-to-end operation by one developer using an OASIS CSV file, local model training, FastAPI inference, browser testing, and optional Gradio deployment.
Clinical boundary: This implementation is a technical baseline and not a clinically validated medical decision system.
