1. Project Overview

NeuroSynth is now a runnable neurological risk prediction system built from real OASIS longitudinal records while preserving the original repository's advanced architecture footprint.

Current practical mission: turn the repository into something you can run daily with predictable behavior: inspect data, train model, serve predictions, validate results, and share a deployable interface.
Boundary: This baseline supports product iteration and demonstration. Clinical production use requires external validation, governance, and regulatory process.
