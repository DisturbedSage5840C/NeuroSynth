1. Project Summary

NeuroSynth is a full-stack research platform for neurological risk prediction using OASIS longitudinal biomarkers. It combines an ensemble biomarker model, an LSTM trajectory model, neural causal discovery, and a structured clinical report generator.

Clinical boundary: this platform is for research and software engineering validation only, not for direct clinical diagnosis.

5. Model Design

Biomarker model

  • RandomForest + GradientBoosting ensemble.
  • Probability is averaged from both classifiers.
  • Risk levels bucketed as Low, Moderate, High, Critical.

Temporal model

  • PyTorch LSTM over patient visit sequences.
  • Outputs 6-step trajectory for months 6..36.
  • Fallback heuristic if sequence training is insufficient.

Causal model

  • Learnable adjacency matrix with acyclicity penalty.
  • Per-variable MLP structural equations.
  • Returns directed edges and intervention estimates.

6. Frontend Features

  • Dashboard with performance and feature charts.
  • Patient analysis form with risk gauge and trajectory plot.
  • Causal network SVG and intervention simulator.
  • Research tab with methodology and references.

7. Local Run Instructions

# backend
source .venv/bin/activate
pip install -r backend/requirements.txt
python -m backend.api

# frontend (new terminal)
cd frontend
npm install
npm run dev -- --host 0.0.0.0 --port 5173

8. Test Checklist

  1. Open frontend: http://localhost:5173
  2. Verify API health: GET /health
  3. Submit sample biomarkers in Patient Analysis tab.
  4. Check trajectory chart and top risk factors.
  5. Run intervention simulation and review deltas.

9. Environment Variables

Use .env for optional report generation token:

HF_TOKEN=your_huggingface_token_here

Never commit real tokens into the repository.

10. Deployment Notes

  • Docker and docker-compose are included for containerized runs.
  • Frontend can be served separately from API in production.
  • Model files are generated in models/ during training/startup.
